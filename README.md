# 📄 Currículo Interativo em Python

Projeto desenvolvido em **Python** que apresenta um **currículo profissional em formato interativo diretamente no terminal**.

O sistema utiliza **menu de navegação**, **efeitos visuais no terminal**, **barra de carregamento** e **organização modular em funções**, proporcionando uma experiência diferenciada para apresentação de informações profissionais.

---

# 🚀 Demonstração

Ao executar o programa, o usuário visualiza um menu interativo:

```
=================================
   CURRÍCULO INTERATIVO PYTHON
=================================

1 - Contato
2 - Habilidades técnicas
3 - Cursos e certificações
4 - Experiência profissional
5 - Objetivo profissional
6 - Formação acadêmica
8 - Exibir currículo completo

0 - Finalizar
```

Cada opção apresenta uma seção do currículo com navegação entre telas.

---

# 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de:

* Demonstrar conhecimentos em **Python**
* Trabalhar com **estrutura de funções**
* Criar **menus interativos em terminal**
* Aplicar **efeitos visuais em CLI**
* Simular uma **aplicação real executável**

Também funciona como uma forma criativa de apresentar um currículo técnico.

---

# 🧠 Funcionalidades

✔ Menu interativo no terminal
✔ Navegação entre seções do currículo
✔ Efeito de texto digitando
✔ Barra de carregamento inicial
✔ Sistema de navegação entre telas
✔ Estrutura modular com funções
✔ Exibição completa do currículo
